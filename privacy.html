Privacy Policy (Политика конфиденциальности)

Приложение: Learn Romanian: Words & Oath

Дата обновления: 2026-01-19

1. Сбор данных

Приложение Learn Romanian: Words & Oath не собирает, не хранит и не передаёт персональные данные пользователей. Данные обучения (например, прогресс и настройки) хранятся только на вашем устройстве.

2. Локальное хранение

Приложение может сохранять на устройстве:

  • Прогресс по словам
  • Выбранный язык интерфейса и дневную цель
  • Статус просмотра onboarding
  • Статус покупки Premium и Oath Pack

Эти данные используются только внутри приложения.

3. Покупки в приложении

Покупки обрабатываются через App Store (Apple). Мы не имеем доступа к вашим платёжным данным. Для проверки статуса покупки и восстановления покупок может использоваться сервис RevenueCat. RevenueCat не передаёт нам ваши платёжные данные и используется только для подтверждения статуса покупки.

4. Аналитика

Приложение не использует рекламные идентификаторы и не показывает рекламу. Мы не используем сторонние сервисы аналитики для отслеживания действий пользователей.

5. Озвучка текста

Функция озвучки может использовать внешний сервис синтеза речи для генерации аудио. В этом случае отправляется только текст слова или фразы, без персональных данных пользователя. Сгенерированное аудио может сохраняться (кэшироваться) на устройстве для ускорения повторного воспроизведения.

6. Права доступа

Приложение не требует специальных разрешений и не запрашивает доступ к камере, микрофону, контактам или геолокации.

7. Изменения политики

Мы можем обновлять эту политику. Актуальная версия всегда доступна по этому адресу.
